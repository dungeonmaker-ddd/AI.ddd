什么是 GitHub Pages？

GitHub Pages 是 GitHub 提供的一项免费的静态网站托管服务。它直接从 GitHub 上的仓库获取 HTML、CSS 和 JavaScript 文件， 通过构建过程运行这些文件，然后发布网站。它是展示开源项目、托管博客或创建作品集的理想选择。

主要特点

  • ✓
    免费托管

    完全免费的静态网站托管服务，无需支付任何费用。

  • ✓
    集成 GitHub

    与 GitHub 仓库完美集成，推送代码后自动部署更新。

  • ✓
    HTTPS 安全

    自动提供 HTTPS 加密，确保网站安全。

  • ✓
    自定义域名

    支持配置自定义域名，让您的网站更专业。

如何使用？

  1. 创建一个新的 GitHub 仓库或使用现有仓库
  2. 在仓库设置中启用 GitHub Pages
  3. 选择部署分支（通常是 main 分支）
  4. 添加网站文件到仓库
  5. 推送更改后，等待几分钟即可在线访问

使用场景

个人博客

创建和维护个人博客，分享知识和经验。

项目文档

托管开源项目的文档和使用说明。

个人作品集

展示个人项目和作品集。

静态网站

托管任何类型的静态网站。
