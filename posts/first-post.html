使用 GitHub Pages 搭建个人博客

2024年1月29日 · 技术分享

今天，我要分享如何使用 GitHub Pages 和 Tailwind CSS 搭建一个简洁美观的个人博客。

为什么选择 GitHub Pages？

  • 完全免费的托管服务
  • 与 Git 版本控制完美集成
  • 支持自定义域名
  • 部署简单快捷

使用 Tailwind CSS 的优势

  • 实用优先的 CSS 框架
  • 高度可定制
  • 响应式设计变得简单
  • 通过 CDN 快速集成
