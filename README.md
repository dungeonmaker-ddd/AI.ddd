# 我的个人博客

这是一个使用 GitHub Pages 搭建的个人博客网站。

## 目录结构

```
.
├── index.html          # 主页
├── styles/            # 样式文件夹
│   └── main.css      # 主要样式文件
├── posts/            # 博客文章目录
└── README.md         # 项目说明文件
```

## 如何使用

1. 克隆此仓库到本地
2. 在 `posts` 目录下添加新的博客文章
3. 修改 `index.html` 以更新最新文章列表
4. 提交更改并推送到 GitHub
5. 访问 `你的用户名.github.io/仓库名` 查看博客

## 本地预览

直接在浏览器中打开 `index.html` 文件即可预览网站效果。
